# 1、需求总述
## 1.1 需求背景
启航平台是中华财险新一代核心系统的员工视角系统门户，承担「应用的发现与访问」入口职能。当前应用信息分散、更新动态与操作指引缺乏统一承载，员工查找系统、了解变更、获取操作指引的成本较高；同时 PMS（项目管理系统）的发布计划与应用侧的更新动态彼此割裂，发布信息无法自动回流到员工视角的门户。

本期重点打通 **PMS 与启航的双向关联**，以「应用」为中枢，把更新日志、操作手册、应用地图整合到统一的员工门户中，形成「启航—知行—聆听」深化应用工具的雏形。

## 1.2 编写目的
明确本期启航平台在应用管理、更新日志、操作手册、新一代应用地图四个模块的功能范围、业务规则、交互要求与验收标准，作为设计、研发、测试与上线验收的统一依据。

## 1.3 名词释义
| 本需求涉及到的常用名词 | 名词解释 | 边界说明 |
| --- | --- | --- |
| 启航平台 | 中华财险新一代核心系统员工视角系统门户，负责应用的发现、访问与信息聚合 | 本期聚焦员工视角门户侧能力 |
| PMS | 项目管理系统，承载各系统的发布窗口、发布单与项目归属 | 启航仅读取并关联 PMS 数据，不在启航内管理发布流程 |
| PMS 发布窗口 | PMS 中定义的发布时间段<br/>在日志管理中用于约束发布类更新日志的归属 | 一条发布类更新日志关联一个发布窗口 |
| PMS 系统属主 | PMS 中某系统的责任方标识，用于应用地图按「属主」检索 | 以 tag 形式挂在应用上，可多选 |
| 应用清单 | 启航侧维护的应用主数据集合（名称、分类、属主等） | 与 PMS 系统通过映射关系关联 |
| 更新日志 | 面向员工的系统变更动态，分「发布类」与「非发布类」 | 发布类需绑定 PMS 发布窗口与所属 PMS 系统 |
| 操作手册 | 面向员工的应用操作指引，按应用独立维护 | 采用左树右文结构，最多 3 层 |
| 新一代应用地图 | 员工/管理人员视角的应用全景视图，支持按名称/属主/分类检索并下钻详情 | 本期新建功能「启航：新一代应用地图」 |
| 应用详情页 | 单个应用的聚合详情页，含操作手册/更新日志/相关课程/我有建议四个 Tab | 相关课程对接知行社，我有建议对接聆听<br/>相关课程和我有建议本期仅留口，不做功能开发 |

# 2、产品需求概述
## 2.1 产品蓝图
<!-- 这是一张图片，ocr 内容为： -->

本期以「应用」为中枢，建立启航与 PMS 的关联，并叠更新日志、操作手册、应用地图三类能力：

+ **关联层**：应用管理支持关联 PMS 系统、标记 PMS 系统属主；应用清单域与 PMS 系统建立映射。
+ **内容层**：每个应用可绑定更新日志（关联发布单）与操作手册（独立目录）。
+ **展现层**：新一代应用地图提供检索与下钻；应用详情页四 Tab 聚合操作手册、更新日志、相关课程（知行）、我有建议（聆听）。
+ **开放层**：更新日志、操作手册均提供对外查询接口，供 PMS、应用内等场景消费。

## 3.4 操作手册
<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/159868/1783504321970-fabc740a-f974-4dbb-b54b-9416fb769f76.png)

操作手册是本期全新模块，为每个应用提供独立的结构化操作指引管理能力。面向三类展示端口：应用内查看（接口/组件）、管能力地图应用详情页查看、PMS 侧（接口消费）。

### 3.4.1 模块说明
每个应用各自维护一套独立的操作手册目录树，应用间互不干扰。目录结构采用**左树右文**编辑界面，最多支持 **3 层**（2 层目录 + 1 层文档），文档内容通过富文本编辑器编写。

本期不涉及版本控制，操作手册修改后直接覆盖发布。

### 3.4.2 大盘列表
管理端新增「操作手册管理 > 手册大盘清单」页面，汇总展示所有应用的操作手册概况。

**页面布局：** 与更新日志查询页面一致的结构——顶部筛选区 + 工具栏 + 数据表格 + 分页。

**表格列定义：**

| 列名 | 宽度 | 说明 |
| --- | --- | --- |

### 3.4.4 业务规则
+ 为每个应用维护一套独立的操作手册目录树，应用间互不干扰。
+ 用户在目录树中点击某个文档节点时，在右侧编辑器加载该文档的标题和内容。
+ 用户点击「保存」时，保存当前编辑内容
+ 用户将 Switch 从 `公开` 拨为 `不公开` 时，在应用详情页隐藏该文档。
+ 用户将 Switch 从 `不公开` 拨为 `公开` 时，在应用详情页展示该文档。
+ 用户粘贴格式文本时，自动识别并转换为富文本 HTML。
+ 用户未保存就离开工作台页面时，提示"有未保存的更改，确定离开吗？"

### 3.4.5 对外展示
**应用详情页「操作手册」Tab：**

+ 展示该应用已发布的操作手册目录树（左侧）和文档内容（右侧）
+ 若该应用无已发布的操作手册，展示"暂无操作手册"空态
+ 目录下无可见文档时，隐藏整个目录

**对外查询接口：**

+ 提供查询接口供 PMS、应用内等场景拉取操作手册
+ 入参至少包含：应用标识
+ 接口返回目录树结构 + 各文档内容
